### Covid-19 Analysis Stack

Sviluppo ambiente analisi flessibile e veloce per analisi Covid-19 (Dati su malattia e distribuzione / erogazione vaccini)

Database: ClickHouse
Hypervisor: Docker 

Per avviare:

```
chmod a+x build.sh
./build.sh
```
